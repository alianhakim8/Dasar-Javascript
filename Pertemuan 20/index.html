Function

Function pada javascript

  • Sebuah Sub program yang dapat dipanggil di bagian lain pada program
  • Merupakan struktur dasar pembentuk dari javascript
  • Kumpulan statement
  • Untuk menggunakan function harus dibuat terlebih dahulu
  • Termasuk First Class Object
Kenapa Function ?
  1. Masalah Reusability ( DRY : Do Not Repeat Yourself )
  2. Dekomposisi
  3. Modularitas
Kategori Function
  • Built in Function

    • Dari javascriptnya sendiri sudah ada tinggal pake saja

  • User Defined Function

    • Menggunakan keyword function
    • Nama Function (optional)
    • Parameter / Argumen
      • Disimpan dalam ()
      • Boleh ada / tidak , boleh lebih dari satu
      • Pisahkan dengan tanda koma (,)
    • Function body dibungkus dengan {}
    • Dapat mengembalikan nilai atau tidak
Membuat Function, Ada dua cara membuat function yaitu :
  • Function Deklarasi
  • Function Ekspresi
Memanggil Function
contoh : Alert(jumlah(1,2));
